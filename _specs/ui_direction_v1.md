# UIビジュアル方針 v1

この文書は、アプリのビジュアル方針を定義する設計資料である。
v0（白背景・簡易Web風）を破棄し、新方針を正本とする。
`_docs/SYSTEM_CORE.md` および `_specs/slot_wheel_spec_v0.md` と矛盾させない。

**バージョン**: v1
**作成日**: 2026-05-24
**適用**: Phase 3 以降の実装・デザイン判断の基準

## 2. 筐体色（確定）

**ピュアアイボリー**

### 基準色

| 部位 | 色の方向 | 補足 |
|---|---|---|
| 筐体メイン | #eeeae2 〜 #e2ddd4 | 白に近いが冷たくない。わずかに黄み |
| 筐体ハイライト（上面） | #ffffff寄り | 半光沢の反射感 |
| 筐体シャドウ（下・側面） | #9e9a90 〜 #96928a | 奥行きと厚みの表現 |
| 筐体ボーダー | #c4bfb4 | 樹脂の縁感 |

### 質感の方向

- 半光沢（マットでも鏡面でもない）
- チープなプラスチックではなく、少しだけ高級感のある樹脂感
- ブラッシュドメタルの光の当たり方は残す（conic-gradientで表現）
- 真っ白にしない。少し黄みがかった「使い込まれた白」

### やらないこと

- 真っ白（冷たくなる）
- グロス鏡面（iPhoneに近づきすぎる）
- グレーに寄せすぎる（シルバー金属感になる）

## 3. LCD（確定）

**暗緑モノクロLCD系**

| 要素 | 方針 |
|---|---|
| LCD背景 | #090b09 （ほぼ黒） |
| アクティブ値の色 | #94b86e 〜 #96b870（暗い緑） |
| アクティブ発光 | text-shadow で控えめに（強すぎない） |
| 非アクティブ値の色 | #2a2e26（暗く落とす） |
| スキャンライン | repeating-linear-gradient で薄く入れる |
| LCD枠 | 筐体色より暗いボーダーで「嵌め込み感」を出す |

### 発光の方針

- 発光は「SF端末の演出」ではなく「古いLCDが光っている」感
- 強すぎない。画面を暗くして見て、ちょうど読める程度
- 青・白・赤の発光は使わない

## 4. ホイール（リング）

| 要素 | 方針 |
|---|---|
| リング外周 | conic-gradientで金属/樹脂の光のムラを表現 |
| リング色温度 | 筐体色に合わせてウォーム寄り（#d4d0c8 〜 #e4e0d6） |
| リング内側トラック | ほぼ黒（#131514）、深い奥行き |
| ノッチ | 12個、3個に1個メジャーノッチ（少し長い） |
| 方向インジケーター | 上下左右に小さな三角。控えめ（opacity 0.3前後） |
| 中央ボタン | 筐体色より少し暗め。border-bottomで押し込み感 |

`wheel_spec_v0.md` の「機械を触っている感覚」「止まる場所が感じられる」を
ビジュアルでも体現する。

## 5. ボタン群

| ボタン | ラベル | 役割 |
|---|---|---|
| MENU | MENU | アクティブスロットを1つ進める |
| 選択 | 選択 | （MVP判断。中央決定と統合可） |
| 生成 → | 生成 → | 現在の組み合わせで記憶生成を開始 |

### ボタンの質感

- border-bottom: 2px で物理的な押し込み感
- :active 時に translateY(1px) + border-bottom縮小
- 筐体と同系色。浮いた別パーツとして見える
- ラベルは Orbitron（英字）または Noto Sans JP（日本語）

---

## 6. タイポグラフィ

| 用途 | フォント | サイズ感 |
|---|---|---|
| アプリ名・ラベル | Orbitron | 極小・スペース広め |
| スロット値（LCD内） | VT323 | ドット系。レトロ感 |
| 日本語ボタン | Noto Sans JP | 細め・読みやすさ優先 |
| ボトムラベル | Orbitron | 極小・装飾的 |

---

## 7. 背景（アプリ外側）

- #111113 〜 #0d0d0f（ほぼ黒）
- 筐体のアイボリーが黒背景で浮かび上がることで「機械として存在感」を出す
- グラデーション・テクスチャは入れない（筐体が主役）

---

## 8. SwiftUI実装への対応

| HTMLプレビューの表現 | SwiftUIでの対応方針 |
|---|---|
| conic-gradient（リング） | AngularGradient |
| linear-gradient（筐体） | LinearGradient |
| box-shadow（立体感） | .shadow() + ZStack重ね |
| border-bottom（ボタン押し込み） | .offset + .shadow調整 |
| repeating-linear-gradient（LCD） | Canvas または重ねView |
| text-shadow（LCD発光） | .shadow(color:, radius:) |

詳細実装はCursor向け指示書に委ねる。
本書はビジュアルの「方向」と「優先順位」のみを固定する。

---

## 9. やらないこと

- 青・白・赤系の発光（SF感が強くなる）
- 真っ黒筐体（前バージョンのダークガンメタル路線）
- グロス鏡面仕上げ
- アニメーションの多用（hover演出・パーティクルなど）
- 筐体以外の背景を装飾する
