# 写真MVP体験仕様 v0

---

## 0. 目的

このMVPでは、AI画像生成APIを使わず、iPhone内処理の写真フィルターだけで「写真を記憶の空気に寄せる体験」が成立するかを確認する。

---

## 1. MVPの中心体験

```
写真を選ぶ / 撮る
↓
レトロ端末UIに写真が読み込まれる
↓
ホイールで年代・時間帯・天気・質感などを選ぶ
↓
写真の色味・粒状感・明るさ・周辺減光が変わる
↓
OKで確定
↓
変換後の写真を保存する
```

---

## 2. 写真入力方法

初期MVPでは、以下の優先順位とする。

| 優先度 | 入力方法 |
|--------|----------|
| 第1優先 | フォトライブラリから選ぶ |
| 第2優先 | カメラで撮る |

**理由**

- フォトライブラリの方が実装確認・比較・テストがしやすい。
- カメラ撮影は後続でもよい。

---

## 3. ホイール調整項目

初期MVPでは、以下の5項目に絞る。

| 項目 | 候補 |
|------|------|
| 年代 | 1998 / 2003 / 2007 / 2012 |
| 季節 | 春 / 夏 / 秋 / 冬 |
| 時間帯 | 朝 / 昼 / 夕方 / 夜 |
| 天気 | 晴れ / 曇り / 雨 / 湿った空気 |
| 質感 | 古いデジカメ / 低彩度フィルム / くすんだ緑 / 夜の室内 / 色あせ |

**場所感について**

場所感は魅力的だが、初期MVPでは外す。

理由：場所感を写真フィルターだけで表現すると限界があり、AI画像変換寄りになりやすいため。

---

## 4. 初期フィルター候補

AIなしでiPhone内処理できる範囲に限定する。

### 1998年 夕方

- 少し低彩度
- 暖色寄り
- 軽い粒状感
- 周辺を少し暗くする

### 2003年 夏

- やや明るい
- 緑を少しくすませる
- コントラストを少し下げる
- 古いデジカメ風の軽いノイズ

### 雨の日の帰り道

- 青緑寄り
- 明るさを少し下げる
- コントラストを下げる
- しっとりした低彩度

### 夜の室内

- 暗め
- 黄色〜緑寄り
- 黒を少し浮かせる
- 粒状感を足す

### 色あせた記憶

- 彩度を下げる
- ハイライトを少し眠くする
- コントラストを弱める
- 周辺減光を弱く入れる

---

## 5. 初期MVPでやらないこと

- AI画像生成API
- 写真の中身の描き換え
- 背景変更
- 人物・建物・空などの生成変更
- サーバー送信
- クラウド保存
- SNS投稿機能
- 課金実装
- 大量プリセット
- 高度な写真編集パネル
